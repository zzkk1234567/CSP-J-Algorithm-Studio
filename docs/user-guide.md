# CSP-J 算法可视化学习系统 - 用户使用手册

## 欢迎使用

欢迎使用CSP-J算法可视化学习系统！本系统专为CSP-J（中国计算机学会非专业级软件能力认证入门级）考试准备而设计，提供15种核心算法的可视化演示和互动学习功能。

## 系统功能介绍

### 1. 算法可视化

系统提供多种算法的动画演示，帮助您直观理解算法执行过程：

- **排序算法**：冒泡排序、选择排序、插入排序、快速排序、归并排序、堆排序
- **搜索算法**：线性搜索、二分搜索
- **图算法**：深度优先搜索(DFS)、广度优先搜索(BFS)、Dijkstra最短路径
- **动态规划**：背包问题、最长公共子序列等

每个算法演示包含：
- 实时动画展示算法执行过程
- 详细的步骤说明（中文）
- 统计数据（比较次数、交换次数、执行时间等）
- 算法复杂度分析

### 2. 互动教程系统

互动教程模式提供步骤式学习体验：
- 分步讲解算法原理
- 互动式问答题目
- 即时反馈和提示
- 知识点关联和扩展

### 3. 性能对比功能

对比不同算法的性能表现：
- 同时运行多个算法
- 实时对比执行速度
- 统计数据并排展示
- 帮助理解算法效率差异

### 4. 自定义数据输入

支持自定义测试数据：
- 手动输入数据序列
- 随机生成数据
- 预设数据模板（已排序、逆序、部分有序等）
- 调整数据规模

### 5. 配置管理

个性化设置选项：
- 主题切换（浅色/深色模式）
- 动画速度调节
- 语言选择（中文/英文）
- 性能监控开关
- 配置自动保存

## 使用说明

### 如何选择算法

1. **浏览算法列表**
   - 打开系统主页，您会看到所有可用算法的卡片
   - 算法按类别分组：排序、搜索、图算法、动态规划

2. **搜索算法**
   - 使用页面顶部的搜索框
   - 输入算法名称关键词进行快速查找

3. **筛选算法**
   - 使用筛选下拉菜单
   - 按难度级别或算法类别筛选

4. **收藏功能**
   - 点击算法卡片上的星标图标收藏
   - 收藏的算法会显示在顶部便于快速访问

5. **最近使用**
   - 系统自动记录您最近学习的算法
   - 最近使用的算法显示在列表顶部

### 控制播放

算法演示提供完整的播放控制：

1. **播放/暂停** (▶️/⏸️)
   - 点击播放按钮开始动画
   - 再次点击暂停当前步骤
   - 快捷键：空格键

2. **单步执行**
   - **下一步** (⏭️)：执行下一个动画步骤
   - **上一步** (⏮️)：返回上一个步骤
   - 快捷键：右箭头键（下一步）、左箭头键（上一步）

3. **重置** (🔄)
   - 重置算法到初始状态
   - 清除所有统计数据
   - 重新生成动画队列

4. **速度调节**
   - 使用速度滑块调整动画播放速度
   - 范围：100ms - 3000ms
   - 实时生效，无需重新开始

5. **进度显示**
   - 进度条显示当前执行进度
   - 显示"步骤 X / 总步数"
   - 点击进度条可跳转到指定步骤

### 自定义数据

1. **手动输入数据**
   - 找到"自定义数据"输入框
   - 输入数字，用逗号分隔（例如：5,2,8,1,9）
   - 点击"应用"按钮

2. **使用数据模板**
   - 点击"数据模板"按钮
   - 选择预设模板：
     - **随机数据**：生成随机数序列
     - **已排序**：生成升序序列
     - **逆序**：生成降序序列
     - **部分有序**：生成部分排序的序列
     - **重复元素**：包含重复值的序列

3. **调整数据规模**
   - 使用"数据量"滑块
   - 范围：5 - 100个元素
   - 注意：数据量过大可能影响性能

4. **数据验证**
   - 系统自动验证输入格式
   - 只允许数字和逗号
   - 数值范围：0-1000
   - 最多1000个元素

### 查看算法信息

1. **算法原理**
   - 点击"算法原理"标签
   - 查看详细的算法说明
   - 包含原理解释和适用场景

2. **代码实现**
   - 点击"代码实现"标签
   - 查看标准C++代码
   - 代码符合CSP-J考试规范
   - 包含详细注释

3. **复杂度分析**
   - 查看时间复杂度（最佳、平均、最坏）
   - 查看空间复杂度
   - 理解算法效率特征

4. **统计数据**
   - 实时显示执行统计
   - 比较次数、交换次数
   - 数组访问次数
   - 执行时间

### 使用互动教程

1. **启动教程**
   - 选择算法后，点击"互动教程"按钮
   - 系统进入教程模式

2. **学习步骤**
   - 按照引导逐步学习
   - 阅读每个步骤的说明
   - 观察对应的可视化演示

3. **回答问题**
   - 教程中会出现思考题
   - 选择您的答案
   - 获得即时反馈

4. **查看提示**
   - 遇到困难时点击"提示"按钮
   - 获取解题思路
   - 不影响学习进度

5. **完成教程**
   - 完成所有步骤后获得完成标记
   - 系统记录学习进度
   - 可随时重新学习

### 个性化设置

1. **打开设置面板**
   - 点击页面右上角的齿轮图标⚙️
   - 设置面板以模态框形式显示

2. **主题设置**
   - 选择浅色或深色主题
   - 立即预览效果
   - 自动保存偏好

3. **动画速度**
   - 全局调整所有算法的默认速度
   - 使用滑块选择合适的速度
   - 范围：100ms - 3000ms

4. **语言选择**
   - 切换界面语言（中文/English）
   - 所有文本自动翻译
   - 包括按钮、标签、提示等

5. **性能监控**
   - 开启/关闭性能指标显示
   - 显示FPS、内存使用、渲染时间
   - 帮助了解系统性能

6. **移动端优化**
   - 自动检测移动设备
   - 手动开启/关闭移动优化
   - 优化触摸交互和布局

## 移动端使用说明

### 手势操作

系统支持触摸手势，提供更便捷的移动端体验：

1. **左滑手势**
   - 在演示区域向左滑动
   - 执行"下一步"操作
   - 快速浏览算法步骤

2. **右滑手势**
   - 在演示区域向右滑动
   - 执行"上一步"操作
   - 返回查看前面的步骤

3. **双击**
   - 双击演示区域
   - 播放/暂停动画
   - 快速控制播放状态

4. **长按**
   - 长按算法卡片
   - 显示快捷菜单
   - 快速收藏或分享

### 布局适配

移动端自动优化布局：

1. **响应式设计**
   - 自动适配不同屏幕尺寸
   - 竖屏和横屏自动调整
   - 保持最佳显示效果

2. **触摸优化**
   - 所有按钮至少44x44像素
   - 增大点击区域
   - 防止误触

3. **简化界面**
   - 隐藏非必要元素
   - 突出核心功能
   - 减少视觉干扰

4. **性能优化**
   - 自动降低动画复杂度
   - 减少DOM操作
   - 提升流畅度

### 移动端最佳实践

1. **横屏观看**
   - 建议横屏使用以获得更好的视野
   - 特别是图算法等复杂可视化

2. **WiFi环境**
   - 首次访问建议使用WiFi
   - 加载所有资源和脚本
   - 后续访问会使用缓存

3. **关闭省电模式**
   - 省电模式可能影响动画流畅度
   - 建议充电时使用

4. **使用手势**
   - 充分利用滑动手势
   - 比点击按钮更快捷
   - 提升学习效率

## 常见问题解答 (FAQ)

### 1. 系统支持哪些浏览器？

系统支持所有现代浏览器：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

建议使用最新版本浏览器以获得最佳体验。

### 2. 为什么动画播放不流畅？

可能的原因和解决方案：
- **数据量过大**：减少数据元素数量（建议50个以内）
- **浏览器标签页过多**：关闭不必要的标签页
- **设备性能不足**：降低动画速度或开启移动端优化
- **后台程序占用**：关闭其他占用资源的程序

### 3. 如何保存我的学习进度？

系统自动保存以下内容到浏览器本地存储：
- 学习过的算法记录
- 收藏的算法列表
- 个人配置偏好
- 教程完成状态

注意：清除浏览器数据会删除这些记录。

### 4. 可以离线使用吗？

首次访问需要联网加载所有资源。加载完成后，系统会缓存资源，支持离线使用核心功能。但以下功能需要联网：
- 错误报告提交
- 在线帮助文档
- 系统更新检查

### 5. 如何重置所有设置？

两种方法：
1. **通过设置面板**：打开设置 → 点击"恢复默认设置"按钮
2. **手动清除**：浏览器设置 → 清除站点数据 → 选择本站点

### 6. 动画速度调节没有效果？

请检查：
- 确保点击了"应用"按钮
- 尝试暂停后重新播放
- 检查是否在教程模式（教程模式速度固定）
- 刷新页面重试

### 7. 自定义数据输入失败？

常见问题：
- **格式错误**：确保只包含数字和逗号，无空格
- **数值超限**：数值应在0-1000范围内
- **数量过多**：最多支持1000个元素
- **特殊字符**：不支持小数点、负号等特殊字符

正确格式示例：`5,2,8,1,9,3`

### 8. 移动端手势不响应？

可能原因：
- **浏览器不支持**：某些旧版浏览器不支持触摸事件
- **滑动距离不够**：需要滑动至少50像素
- **垂直滑动**：只识别水平滑动手势
- **优化未开启**：检查设置中的移动端优化选项

### 9. 如何查看错误日志？

步骤：
1. 打开设置面板（齿轮图标）
2. 点击"查看错误日志"按钮
3. 查看详细的错误记录
4. 可以复制日志用于反馈问题

### 10. 系统支持哪些算法？

当前支持15种算法：

**排序算法（6种）**
- 冒泡排序、选择排序、插入排序
- 快速排序、归并排序、堆排序

**搜索算法（2种）**
- 线性搜索、二分搜索

**图算法（3种）**
- 深度优先搜索(DFS)
- 广度优先搜索(BFS)
- Dijkstra最短路径

**动态规划（4种）**
- 0-1背包问题
- 最长公共子序列(LCS)
- 最长递增子序列(LIS)
- 编辑距离

更多算法持续添加中！

### 11. 如何反馈问题或建议？

我们欢迎您的反馈：
- 通过错误日志系统报告技术问题
- 在设置面板中找到"反馈"选项
- 描述问题或建议的详细信息
- 我们会及时查看并改进

### 12. 性能监控显示什么信息？

性能监控面板显示：
- **FPS**：每秒帧数，衡量动画流畅度（建议30+）
- **内存使用**：JavaScript堆内存占用（MB）
- **渲染时间**：每帧渲染耗时（ms）
- **动画时间**：动画执行总时长

这些指标帮助您了解系统性能状态。

## 学习建议

1. **循序渐进**：从简单算法（冒泡排序）开始，逐步学习复杂算法
2. **反复观看**：多次观看动画，理解每个步骤的含义
3. **动手实践**：尝试自定义不同的数据，观察算法行为
4. **对比学习**：使用性能对比功能，理解算法效率差异
5. **完成教程**：认真完成互动教程，巩固知识点
6. **编写代码**：参考代码实现，自己动手编写
7. **做题练习**：结合CSP-J真题，应用所学知识
