## 🎯 升级指南

### 从 v1.0.0 升级

1. **备份数据**（可选）
   ```javascript
   // 导出学习进度
   const progress = window.progressTracker.exportProgress();
   // 保存到文件
   ```

2. **更新文件**
   - 下载最新版本
   - 替换所有文件
   - 保留用户数据（localStorage自动保留）

3. **清除缓存**
   - 刷新浏览器缓存（Ctrl+F5）
   - 确保加载最新资源

4. **验证功能**
   - 打开主页面
   - 测试核心功能
   - 检查配置是否保留

### 新用户安装

1. **下载项目**
   ```bash
   git clone https://github.com/your-username/cspj-algorithm-visualizer.git
   cd cspj-algorithm-visualizer
   ```

2. **启动服务器**
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx http-server -p 8000
   ```

3. **访问应用**
   ```
   http://localhost:8000/csp-j-learning-tool.html
   ```
