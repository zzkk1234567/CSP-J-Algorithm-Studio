## 🚀 使用方法

### 快速开始
1. 打开 `final-complete-demo.html` 获得完整体验
2. 选择算法类别（排序、搜索、图、动态规划）
3. 选择具体算法并观看可视化演示
4. 使用键盘快捷键控制演示过程
5. 点击"互动教程"进行深入学习
6. 使用"运行测试"验证系统功能

### 键盘快捷键
- **空格键**：播放/暂停演示
- **→键**：单步执行
- **Ctrl+R**：重置演示
- **Ctrl+F**：切换全屏
- **1-5键**：快速设置动画速度
- **Ctrl+Shift+P**：显示性能监控
